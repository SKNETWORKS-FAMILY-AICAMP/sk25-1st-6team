# 3. 프로젝트 개요

## 📕 프로젝트명
Vehicle AirInsight

## ✅ 배경 및 목적
차량 등록 대수의 지속적인 증가는 대기질 악화의 주요 원인입니다. 본 프로젝트는 **일반 내연기관 차량 증가와 대기오염도 간의 상관관계를 시각적으로 증명**하여 경각심을 주고, 사용자가 **무공해차(전기차, 수소차)로 쉽게 전환**할 수 있도록 실질적인 정보(보조금, 충전소)를 제공하는 것을 목적으로 합니다.

## 🖐️ 프로젝트 소개
1.  **데이터 시각화:** 전국 시/도별 자동차 등록 현황과 대기질 수치를 지도 및 차트로 비교 분석.
2.  **무공해차 전환 가이드:** 복잡한 보조금 계산을 자동화하고, GPS/IP 기반으로 내 주변 충전소를 탐색.
3.  **맥락 기반 정보 제공:** 사용자가 선택한 차량 타입(승용/화물 등)과 용도(자가용/영업용)에 맞는 FAQ를 자동 필터링하여 제공.

## ❤️ 기대효과
본 프로젝트는 단순한 데이터 나열을 넘어, **데이터 시각화를 통한 문제 인식**과 **실질적인 행동 유도(무공해차 전환)**를 목표로 합니다.

*   **🔍 데이터 기반의 환경 인식 제고:**
    *   직관적인 지도와 히트맵 시각화를 통해 일반 사용자가 **차량 증가와 대기 오염의 상관관계**를 한눈에 파악할 수 있어, 환경 문제에 대한 경각심을 높입니다,.
*   **🌱 친환경 차량 전환 장벽 해소:**
    *   복잡한 지자체별/차종별 보조금 계산 과정을 자동화하여, 전기·수소차 구매를 고민하는 사용자에게 **경제적 혜택을 명확하게 제시**합니다.
    *   GPS 기반 충전소 찾기 기능을 통해 무공해차 운전자의 가장 큰 고민인 **'충전 인프라 불안(Range Anxiety)'을 완화**하여 친환경차 보급 활성화에 기여합니다.
*   **⚡ 정보 습득의 효율성 극대화:**
    *   사용자의 선택 흐름(지역 → 차종 → 용도)에 맞춰 **관련된 FAQ만 큐레이션**하여 보여줌으로써, 방대한 정보 속에서 필요한 내용을 찾는 시간을 획기적으로 단축합니다,.

## 👤 대상 사용자
이 서비스는 환경 데이터에 관심이 있는 일반인부터 실제 차량 구매를 앞둔 소비자까지 폭넓게 활용할 수 있습니다.

1.  **🚙 예비 무공해차 구매자 (Prospective Buyers):**
    *   전기차나 수소차 구매를 고려 중이지만, **"내가 받을 수 있는 정확한 보조금"**이 얼마인지 궁금한 사용자.
    *   내 거주지 주변의 충전 인프라 현황을 미리 확인하고 싶은 사용자.
2.  **🔌 기존 전기·수소차 운전자 (EV/FCEV Drivers):**
    *   운행 중 내 위치 주변의 **가까운 충전소 정보**를 빠르게 찾고자 하는 사용자.
    *   차량 유지보수나 보증 수리 등 특정 차종(승용/화물)에 맞는 관리 정보를 얻고 싶은 사용자.
3.  **📊 데이터 분석가 및 환경 연구자 (Data Analysts & Researchers):**
    *   전국 시/도별 자동차 등록 현황과 대기질 데이터의 **시계열 추이 및 지역별 분포**를 분석하고자 하는 전문가 및 학생,.
4.  **🌏 환경 문제에 관심 있는 일반 시민 (General Public):**
    *   우리 동네의 자동차 등록 대수와 대기 오염도가 다른 지역에 비해 어느 정도 수준인지 직관적으로 확인하고 싶은 사용자.
